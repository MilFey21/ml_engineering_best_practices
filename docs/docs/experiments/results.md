# Результаты экспериментов

Этот раздел содержит результаты проведенных экспериментов и инструкции по генерации отчетов.

## Запуск экспериментов

Для запуска всех экспериментов:

```bash
pixi run churn-experiments
```

## Просмотр результатов

### Через ClearML UI

1. Запустите ClearML Server:
   ```bash
   pixi run clearml-server-start
   ```

2. Откройте http://localhost:8080

3. Перейдите в **Projects → Churn Prediction**

### Через командную строку

```bash
# Список моделей
pixi run churn-model-registry

# Сравнение моделей
pixi run churn-model-registry compare --metric-name test_f1_score --top-n 5
```

## Метрики

Все эксперименты логируют следующие метрики:

- `test_accuracy` - точность на тестовой выборке
- `test_f1_score` - F1-Score на тестовой выборке
- `test_precision` - Precision на тестовой выборке
- `test_recall` - Recall на тестовой выборке
- `test_roc_auc` - ROC-AUC на тестовой выборке

## Автоматическая генерация отчетов

Отчеты автоматически генерируются с графиками и визуализациями:

```bash
# Генерация отчета с графиками (по умолчанию)
pixi run generate-report

# Генерация без графиков
pixi run generate-report --no-plots

# Генерация всех отчетов
pixi run generate-all-reports
```

### Структура отчетов

Отчеты сохраняются в `reports/experiments/` и включают:

- **Таблицы сравнения метрик** - топ эксперименты по каждой метрике
- **Объединенные графики метрик** - визуализация всех метрик одного эксперимента
- **Confusion matrix** - матрица ошибок для каждого эксперимента
- **Дополнительные графики** - графики обучения и другие визуализации

### Структура файлов

```
reports/
  experiments/
    experiment_report.md
    plots/
      task_<task_id>/
        combined_metrics.png
        confusion_matrix.png
        ...
```

## Сравнение экспериментов

Отчеты включают автоматическое сравнение экспериментов:

- Сортировка по метрикам (F1-Score, Accuracy, ROC-AUC и т.д.)
- Таблицы с топ-N экспериментами
- Визуализация метрик для лучших экспериментов
- Фильтрация по уникальным именам экспериментов
