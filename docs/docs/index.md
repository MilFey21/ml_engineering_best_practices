# ml_engineering_best_practices

Проект для предсказания оттока клиентов с использованием лучших практик машинного обучения.

## Быстрый старт

См. раздел [Getting Started](getting-started.md) для инструкций по установке и началу работы.

## Документация

- **[Getting Started](getting-started.md)** - установка и быстрый старт
- **[API Documentation](api/overview.md)** - документация API
  - [Models API](api/models.md) - модули для работы с моделями
  - [Utils API](api/utils.md) - вспомогательные утилиты
- **[Experiments](experiments/results.md)** - результаты экспериментов
- **[Reproducibility](reproducibility.md)** - инструкции по воспроизведению результатов

## Основные возможности

- Обучение моделей машинного обучения (Random Forest, Gradient Boosting, Logistic Regression)
- Управление экспериментами с помощью MLflow и ClearML
- Версионирование данных с помощью DVC
- Автоматическая генерация документации
- Конфигурация через Hydra
