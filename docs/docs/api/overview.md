# API Overview

Документация API проекта предсказания оттока клиентов.

## Модули

- **[Models](models.md)** - API для обучения и использования моделей машинного обучения
- **[Utils](utils.md)** - Вспомогательные утилиты и функции

## Автоматическая генерация документации

Документация API автоматически генерируется из docstrings исходного кода с помощью [mkdocstrings](https://mkdocstrings.github.io/).

Все функции, классы и методы документированы в соответствии с Google Style docstrings.
