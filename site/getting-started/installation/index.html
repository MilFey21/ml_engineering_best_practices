Установка и настройка

Это руководство поможет вам установить и настроить проект для предсказания оттока клиентов.

Требования

  • Python 3.12+
  • Git
  • Pixi (рекомендуется) или pip/conda

Установка Pixi

Windows

# Через winget
winget install prefix-dev.pixi

# Или через PowerShell
iwr https://pixi.sh/install.ps1 | iex

Linux/Mac

curl -fsSL https://pixi.sh/install.sh | bash

Клонирование репозитория

git clone https://github.com/your-username/ml_engineering_best_practices.git
cd ml_engineering_best_practices

Установка зависимостей

Использование Pixi (рекомендуется)

# Установка всех зависимостей
pixi install

# Активация окружения
pixi shell

Альтернатива: pip

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

Настройка Kaggle API

Для загрузки датасета необходимо настроить Kaggle API credentials:

  1. Создайте аккаунт на Kaggle (если еще нет)
  2. Перейдите в Settings → API → Create New Token
  3. Скачайте kaggle.json
  4. Поместите файл в:
  5. Windows: C:\Users\<username>\.kaggle\kaggle.json
  6. Linux/Mac: ~/.kaggle/kaggle.json

Важно: Установите правильные права доступа:

# Linux/Mac
chmod 600 ~/.kaggle/kaggle.json

Альтернатива: Используйте переменные окружения:

export KAGGLE_USERNAME=your_username
export KAGGLE_KEY=your_key

Настройка ClearML (опционально)

Для использования ClearML для отслеживания экспериментов:

  1. Запустите локальный ClearML Server: bash pixi run clearml-server-start

  2. Настройте credentials: bash clearml-init

  3. API Server: http://localhost:8008
  4. Web UI: http://localhost:8080
  5. Access Key и Secret Key: получите из Web UI (http://localhost:8080 → Settings → Workspace)

Подробнее см. Руководство по развертыванию.

Настройка Pre-commit Hooks

Для автоматической проверки качества кода:

pixi run setup
# или
pre-commit install

Проверка установки

Проверьте, что все установлено правильно:

# Проверка версии Python
python --version

# Проверка установки зависимостей
pixi run python -c "import pandas, sklearn, clearml; print('All dependencies installed!')"

# Проверка структуры проекта
pixi run python -c "from src.config import PROJ_ROOT; print(f'Project root: {PROJ_ROOT}')"
