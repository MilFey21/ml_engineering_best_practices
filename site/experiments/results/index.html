Результаты экспериментов

Этот раздел содержит результаты проведенных экспериментов.

Запуск экспериментов

Для запуска всех экспериментов:

pixi run churn-experiments

Структура результатов

Результаты экспериментов сохраняются в:

  • ClearML - если настроен ClearML Server
  • MLflow - если используется MLflow
  • Локальные файлы - в директории reports/

Просмотр результатов

Через ClearML UI

  1. Запустите ClearML Server: bash pixi run clearml-server-start

  2. Откройте http://localhost:8080

  3. Перейдите в Projects → Churn Prediction Experiments

Через командную строку

# Список моделей
pixi run churn-model-registry

# Сравнение моделей
pixi run churn-model-registry compare --metric-name test_f1_score --top-n 5

Метрики

Все эксперименты логируют следующие метрики:

  • test_accuracy - точность на тестовой выборке
  • test_f1_score - F1-Score на тестовой выборке
  • test_precision - Precision на тестовой выборке
  • test_recall - Recall на тестовой выборке
  • test_roc_auc - ROC-AUC на тестовой выборке

Визуализации

Графики и визуализации результатов доступны в:

  • ClearML UI - confusion matrix, метрики обучения, все визуализации
  • Отчетах в формате Markdown - автоматически скачиваются и встраиваются в отчеты

Автоматическое добавление графиков в отчеты

При генерации отчетов скрипт автоматически:

  1. Скачивает графики из ClearML задач для топовых экспериментов
  2. Создает визуализации скалярных метрик (графики обучения)
  3. Сохраняет изображения (confusion matrix и другие) локально
  4. Встраивает графики в markdown-отчеты

Структура файлов:

reports/
  experiments/
    experiment_report.md
    plots/
      task_<task_id>/
        Metrics_test_f1_score.png
        Metrics_test_accuracy.png
        Plots_Confusion_Matrix.png

Опции генерации отчетов:

# Генерация отчета с графиками (по умолчанию)
pixi run generate-report

# Генерация без графиков
pixi run generate-report --no-plots

# Указать кастомную папку для графиков
pixi run generate-report --plots-dir reports/experiments/custom_plots

Автоматическая генерация отчетов

Отчеты автоматически генерируются при запуске экспериментов. См. раздел Воспроизводимость для подробностей.
